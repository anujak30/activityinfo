Bonnes nouvelles! Vous pouvez utiliser ActivityInfo même lorsque vous n'avez pas une connexion internet.

Activer le mode hors-ligne vous permettra de continuer à travailler avec ActivityInfo même si un connexion internet n'est pas disponible. Quand une connexion est disponible, sera ActivityInfo synchroniser avec le serveur.

Pour ouvrir ActivityInfo en mode hors connexion, vous pouvez simplement taper www.activityinfo.org dans votre navigateur la barre d'adresse.

Nota Bena: si vous activez le mode hors ligne, ActivityInfo va stocker des données localement sur votre ordinateur, ce qui pourrait être accessible à toute personne utilisant cet ordinateur. Vous ne devriez pas activer le mode hors ligne sur public ou partagé ordinateurs.
